Lesson 1. Basics of the Web and HTML

1. How The Web Works

The Web consists of several compuetrs and servers that work together to allow us to surf the web.When we type in a Url like www.udacity.com ,our computer sends a request known as an HTTP request , to a server . The server then finds the HTML document or file and sends it back to our computer.

2. HTML

HTML is Hypertext Markup Language. This consists of text content"visual",markup,"what it looks like",and references, images, and links to other pages,etc... Some componets of HTML are tags,elements,and attributes. It is not important to remember all of the tags because simply "Programmers can't remember everything". You can easiy reference them by google of course.

3.Coding

Coding works when programmers write text in a language that a computer understands. Computers are stupid, so programmers need to write excatly the way a computer understands (also known as the correct syntax).

Lesson 2. Creating a Structured Document in HTML

1.HTML and CSS are both "languages".HTML controls the structure of a page while CSS controls the style of a page.

2.Tree Like Structure

Tree like structures are formed from the content within different tags. These structures can be found in the body or heading og a page and can contain different tags under other tags such as a p tag under a div tag ,etc...

3.Boxes
All content displayed on a webpage are in Boxes.

Lesson 3: Adding CSS to HTML

1. Avoid Repetition

Avoiding repetition can bring consistency to your web page and help reduce errors made when programming.This makes it easy to make changes, often programmers will do the copy and paste method and then when they make a change to one segment it will create errors throughout the page unless you edit everywhere the repeated code is.

2.CSS

CSS stands for Cascading Style Sheets.CSS controls HTML by rules. Everything between the curly braces will be interperted as a rule that should be applied to every Tag element such as a div or span element.
